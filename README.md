# 🤖 MoroGPT

Un asistent AI rapid, securizat și privat, care rulează local.

### ✨ Funcții Premium:
- 🚀 **Streaming în timp real:** Răspunsurile apar cuvânt cu cuvânt (zero timp de așteptare).
- 🖼️ **Vision Support:** Încarcă imagini pentru a fi analizate de AI.
- 🧪 **LaTeX Rendering:** Formule matematice și științifice afișate impecabil.
- 🌑 **Design Modern:** Interfață Dark Mode cu Glassmorphism și auto-scroll.
- 🔒 **100% Privat:** Folosește LM Studio local.

### 🛠️ Tehnologii folosite:
- React (Vite)
- KaTeX (Matematică)
- React-Markdown
- Local LLM API (LM Studio)

### 🚀 Cum îl pornești:
1. Pornit **LM Studio** și încarcă un model (ex: Qwen2-VL pentru imagini).
2. Pornește Local Server în LM Studio pe portul `1234`.
3. Rulează în terminal: `npm install` și apoi `npm run dev`.
